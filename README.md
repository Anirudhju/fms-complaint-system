# FMS Complaint Management System

A full-stack complaint management system built with Angular, Node.js, Express, and Microsoft SQL Server.

## Features

- User authentication and role-based access
- Complaint creation with optional image upload
- Department-based complaint assignment
- Complaint status tracking
- Complaint history logs
- Feedback submission after resolution
- Admin and staff management panel

## Tech Stack

Frontend:
- Angular
- TypeScript
- HTML / CSS

Backend:
- Node.js
- Express

Database:
- Microsoft SQL Server

## System Roles

User
- Create complaints
- Upload complaint images
- Submit feedback

Staff
- View department complaints
- Update complaint status

Admin
- Manage users
- Manage departments
- View all complaints

## Project Structure
fms-complaint-system
├── backend

## Installation

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
